# 血压趋势的解读方法

## 为什么看趋势比看单点更有价值

### 血压的波动性

血压是动态变化的生理指标，受多种因素影响：

- **生理因素**：情绪、活动、饮食、睡眠、体位
- **环境因素**：温度、噪音、光线
- **测量因素**：设备误差、操作不规范

单次血压波动可达±10-15 mmHg，仅凭单次测量判断血压水平容易误判。

### 趋势的价值

趋势分析能：
- 平滑短期波动，反映真实血压水平
- 发现血压变化规律（晨峰、夜间下降）
- 评估治疗效果
- 早期预警血压恶化

## 观察窗口的选择

| 观察窗口 | 适用场景 |
|---------|---------|
| 7天 | 判断近期是否异常波动 |
| 14天 | 评估生活方式/用药调整后的效果 |
| 30天 | 月度复盘，与前月对比 |
| 90天 | 判断长期控制是否达标 |

## 核心趋势指标

### 平均血压

- **收缩压均值**：反映整体收缩压水平
- **舒张压均值**：反映整体舒张压水平
- **计算方法**：去除第1天后，计算连续测量天数的平均值

### 达标率

- **定义**：血压达到目标值的测量次数占总测量次数的比例
- **目标值**：家庭血压<135/85 mmHg
- **意义**：达标率≥80%提示控制良好

### 血压波动幅度

- **最大值-最小值**：反映血压波动范围
- **标准差**：反映血压离散程度
- **意义**：波动过大提示血压不稳定，心血管风险增加

### 时段差异

- **清晨血压**：起床后1小时内测量
- **夜间血压**：睡前测量
- **意义**：清晨血压过高提示晨峰控制不佳

## 如何识别"控制良好"

### 诊断标准

1. 近4周家庭血压均值<135/85 mmHg
2. 达标率≥80%
3. 波动幅度收窄（标准差<10 mmHg）
4. 清晨血压不超过全天均值15 mmHg
5. 无反复极端值（≥180或≤90 mmHg）

### 表现特征

- 血压曲线平稳
- 晨峰不高
- 夜间下降正常（10-20%）
- 无明显时段差异

## 如何识别"正在恶化"

### 警示信号

1. **均值上升**
   - 近7日均值比前7日均值上升>5%
   - 收缩压均值上升>5 mmHg

2. **达标率下降**
   - 从稳定>80%跌到<60%
   - 连续3天以上未达标

3. **波动加大**
   - 标准差明显增加
   - 出现极端值（≥180或≤90 mmHg）

4. **清晨血压异常**
   - 清晨血压持续高于全天均值10 mmHg以上
   - 清晨血压≥145/90 mmHg

5. **夜间血压异常**
   - 夜间血压下降不足10%（非杓型）
   - 夜间血压高于白天（反杓型）

### 需要就医的情况

- 连续7天均值超过目标值10 mmHg以上
- 达标率持续<60%
- 出现血压≥180/110 mmHg伴症状
- 血压波动明显加大

## 如何识别"正在改善"

### 改善信号

1. 均值持续下降
2. 达标率上升（从<60%升至>80%）
3. 波动幅度收窄
4. 清晨血压下降
5. 夜间下降恢复正常

### 评估时机

- 生活方式干预：14天后评估
- 药物调整：2-4周后评估
- 长期效果：90天后评估

## 趋势报告的内容

一份完整的趋势报告应包含：

1. **基本信息**
   - 观察窗口（天数）
   - 测量次数
   - 测量天数

2. **核心指标**
   - 收缩压均值、舒张压均值
   - 收缩压最大值、最小值
   - 舒张压最大值、最小值
   - 达标率

3. **时段分析**
   - 清晨血压均值
   - 夜间血压均值
   - 时段差异

4. **趋势对比**
   - 与前一周对比
   - 与前一月对比

5. **异常提示**
   - 极端值标注
   - 波动过大警告
   - 达标率过低警告

## 智能助手的作用

智能健康助手可以：

- 自动计算各观察窗口的均值、极值、达标率
- 高亮异常时段与偏离度
- 生成趋势对比报告
- 提醒用户测量时间
- 提示就医时机

**边界**：
- 不提供具体药物调整建议
- 不替代医生诊断
- 异常提醒仅供参考，最终决策由医生作出
