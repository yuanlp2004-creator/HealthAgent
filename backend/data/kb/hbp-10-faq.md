# 高血压常见问题与答疑

## Q1. 测血压时两只胳膊读数不一样，哪只算数？

**答**：首次测量建议双臂都测，以较高一侧为准。两侧差异>10 mmHg提示外周血管疾病可能，需告知医生。之后固定测量较高一侧。

## Q2. 血压高就一定要吃药吗？

**答**：不一定。是否需要药物治疗取决于血压水平和心血管风险：

| 情况 | 建议 |
|------|------|
| 1级高血压 + 低危 | 生活方式干预1-3个月，未达标再考虑药物 |
| 1级高血压 + 中危 | 生活方式干预1个月，未达标启动药物 |
| 1级高血压 + 高危/很高危 | 立即启动药物治疗 |
| 2级及以上高血压 | 立即启动药物治疗 |

最终方案由医生评估后决定。

## Q3. 降压药需要终生服用吗？

**答**：绝大多数患者需要长期甚至终生服药。少数患者在以下情况下可能减量或停药：

- 显著减重（>10kg）
- 戒酒、改善生活方式
- 经医生评估血压长期稳定达标

**注意**：擅自停药是血压反弹和心脑血管事件的常见原因。任何药物调整必须在医生指导下进行。

## Q4. 一天测几次血压合适？

**答**：

| 情况 | 测量频率 |
|------|---------|
| 初诊/调整方案期 | 每天早晚各1次，连续7天 |
| 血压稳定达标期 | 每周1-2天，早晚各1次 |
| 出现症状时 | 额外加测，记录时间与诱因 |

## Q5. 为什么运动后血压反而偏低？

**答**：中低强度运动后1-3小时会出现"运动后低血压"（post-exercise hypotension），收缩压可下降5-15 mmHg。这是正常的生理反应，测量血压应选择静息30分钟以上的状态。

## Q6. 每次去医院测都偏高，在家测就正常，是不是没事？

**答**：这可能是"白大衣高血压"。虽然不是持续性高血压，但研究提示：

- 未来进展为持续性高血压的风险高于血压完全正常者
- 心血管风险介于正常血压和持续性高血压之间

建议定期监测，每年至少一次动态血压评估。

## Q7. 每天早上第一次测量的血压特别高，正常吗？

**答**：清晨血压本身是一天中的第一个高峰（晨峰现象）。但如果：

- 起床后1小时内血压持续≥135/85 mmHg
- 清晨血压高出全天均值10 mmHg以上

应告知医生，评估晨峰控制情况，可能需要调整用药时间或药物种类。

## Q8. 心率偶尔跳到120怎么办？

**答**：

| 情况 | 建议 |
|------|------|
| 偶发（数十秒内）+ 无症状 | 可能与情绪、咖啡因、体位有关，可观察 |
| 持续数分钟以上 + 无诱因 | 建议就医评估 |
| 伴胸闷、头晕、乏力 | 需就医行心电图检查 |
| 静息心率持续>100次/分钟 | 需就医排查原因 |

## Q9. 血压降到多少算达标？

**答**：

| 人群 | 诊室血压目标 | 家庭血压目标 |
|------|-------------|-------------|
| 一般高血压患者 | <140/90 mmHg，能耐受可<130/80 mmHg | <135/85 mmHg，能耐受可<125/75 mmHg |
| 65-79岁老年人 | <140/90 mmHg，能耐受可<130/80 mmHg | <135/85 mmHg |
| ≥80岁老年人 | <150/90 mmHg，能耐受可<140/90 mmHg | <145/90 mmHg |
| 糖尿病/慢性肾病 | <130/80 mmHg | <125/75 mmHg |
| 冠心病 | <130/80 mmHg，舒张压不低于60 mmHg | <125/75 mmHg |

## Q10. 降压药什么时候吃最好？

**答**：

- **长效降压药**：通常早晨起床后服用，覆盖全天
- **清晨高血压者**：可考虑睡前服用长效药物
- **短效降压药**：按医生指导的时间服用

具体服药时间应遵医嘱，不要自行调整。

## Q11. 血压正常了可以停药吗？

**答**：不可以。血压正常是药物控制的结果，停药后血压会反弹。血压稳定达标后，医生可能会考虑减量，但需严密监测。擅自停药是危险的。

## Q12. 降压药有副作用吗？

**答**：任何药物都可能有副作用，但：

- 大多数副作用轻微、可逆
- 降压药的获益远大于风险
- 出现不适及时告知医生，可调整药物

常见副作用：
- ACEI/ARB：干咳（ACEI）、血钾升高
- CCB：面部潮红、下肢水肿
- 利尿剂：低钾、血糖/尿酸升高
- β受体阻滞剂：心动过缓、乏力

## Q13. 高血压能预防吗？

**答**：可以。预防措施包括：

1. 限盐（<5g/天）
2. DASH饮食模式
3. 控制体重（BMI 18.5-23.9 kg/m²）
4. 规律运动（每周≥150分钟）
5. 戒烟限酒
6. 心理平衡
7. 定期监测血压（35岁以上每年至少一次）

## Q14. AI健康助手能替代医生吗？

**答**：不能。AI助手的作用是：

- 数据管理：记录、存储血压数据
- 趋势分析：计算均值、达标率
- 常识科普：提供高血压基础知识
- 就医提醒：提示异常情况、提醒就医

**边界**：
- 不提供诊断
- 不提供处方建议
- 不提供用药调整建议
- 涉及诊断和治疗决策必须由医生作出
