# 高血压的分级与心血管风险分层

## 血压水平分级

根据诊室血压水平，将血压分为以下类别：

| 类别 | 收缩压（mmHg） | 舒张压（mmHg） |
|------|---------------|---------------|
| 正常血压 | <120 | <80 |
| 正常高值 | 120-139 | 80-89 |
| 高血压1级（轻度） | 140-159 | 90-99 |
| 高血压2级（中度） | 160-179 | 100-109 |
| 高血压3级（重度） | ≥180 | ≥110 |
| 单纯收缩期高血压 | ≥140 | <90 |

**注**：当收缩压和舒张压分属不同级别时，以较高的分级为准。

## 心血管风险分层

高血压患者的心血管总体风险分为低危、中危、高危和很高危四个层次。分层依据包括：

### 影响风险的因素

1. **心血管危险因素**
   - 年龄（男性≥55岁，女性≥65岁）
   - 吸烟
   - 血脂异常（总胆固醇≥5.7mmol/L或LDL-C≥3.3mmol/L）
   - 早发心血管病家族史（一级亲属发病年龄<50岁）
   - 肥胖（BMI≥28）或腹型肥胖（腰围男性≥90cm，女性≥85cm）
   - 高同型半胱氨酸血症

2. **靶器官损害**
   - 左心室肥厚（心电图或超声心动图诊断）
   - 颈动脉内膜中层增厚（IMT≥0.9mm）或斑块
   - 肾功能受损（eGFR 30-59 ml/min/1.73m²）
   - 微量白蛋白尿（30-300mg/24h）

3. **伴发临床疾病**
   - 脑血管病（缺血性卒中、脑出血、短暂性脑缺血发作）
   - 心脏病（心肌梗死、心绞痛、冠脉血运重建、心力衰竭）
   - 肾脏病（糖尿病肾病、肾功能不全）
   - 外周血管疾病
   - 视网膜病变（出血、渗出、视乳头水肿）
   - 糖尿病

### 风险分层表

| 其他危险因素和病史 | 1级高血压 | 2级高血压 | 3级高血压 |
|------------------|----------|----------|----------|
| 无 | 低危 | 中危 | 高危 |
| 1-2个危险因素 | 中危 | 中危 | 很高危 |
| ≥3个危险因素或靶器官损害 | 高危 | 高危 | 很高危 |
| 伴发临床疾病或糖尿病 | 很高危 | 很高危 | 很高危 |

## 风险分层的临床意义

风险分层决定了治疗的紧迫性和目标值：

- **低危**：可先进行生活方式干预数月，血压未达标再启动药物治疗
- **中危**：生活方式干预1-3个月，血压未达标启动药物治疗
- **高危/很高危**：立即启动药物治疗

## 对自测血压的解读建议

家庭自测血压发现单次升高不必过度紧张。建议：

1. 连续监测7天，排除测量误差
2. 记录早晚血压变化趋势
3. 若家庭血压持续≥135/85 mmHg，建议就医评估
4. 若血压≥180/110 mmHg伴头痛、胸闷等症状，应立即就医
