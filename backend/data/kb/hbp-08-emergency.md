# 高血压急症与血压异常的识别处理

## 高血压急症

### 定义

血压突然显著升高（通常≥180/120 mmHg），**伴进行性靶器官损害**。

### 常见类型

1. **高血压脑病**
   - 剧烈头痛、恶心呕吐
   - 意识模糊、抽搐、昏迷
   - 视乳头水肿、视网膜病变

2. **急性左心衰竭**
   - 呼吸困难、端坐呼吸
   - 咳粉红色泡沫痰
   - 肺部湿啰音

3. **急性冠脉综合征**
   - 剧烈胸痛、胸闷
   - 心电图ST-T改变

4. **主动脉夹层**
   - 突发撕裂样胸背痛
   - 双上肢血压不对称

5. **脑卒中**
   - 肢体麻木、无力
   - 言语不清、口角歪斜
   - 意识障碍

6. **子痫/重度子痫前期**
   - 头痛、视物模糊
   - 抽搐（子痫）

### 处理原则

**立即就医或拨打急救电话（120）**

- 收缩压≥180 mmHg和/或舒张压≥120 mmHg
- 伴上述任一靶器官损害症状
- 不要自行服药
- 保持安静、平卧
- 监测血压、心率

## 高血压亚急症

### 定义

血压显著升高（≥180/120 mmHg），**无急性靶器官损害**。

### 处理原则

1. 安静休息，消除紧张因素
2. 5分钟后复测血压
3. 若仍≥180/120 mmHg：
   - 联系医生或尽快就医（数小时内）
   - 不要自行加倍服药
4. 若复测下降，继续监测

### 注意事项

- 区分急症与亚急症的关键是**有无靶器官损害症状**
- 单纯血压升高无靶器官损害者，通常不需紧急降压
- 过快降压可能导致脑灌注不足

## 血压显著升高但无症状

### 情况判断

血压≥180/120 mmHg，但无头痛、胸痛、呼吸困难、肢体麻木等症状。

### 处理建议

1. 安静休息5-10分钟
2. 排除干扰因素：
   - 刚运动、情绪激动
   - 憋尿、喝浓咖啡
   - 测量误差
3. 复测2-3次，间隔1-2分钟
4. 若复测仍高：
   - 尽快预约门诊（数小时至1天内）
   - 不要自行调整药物
5. 记录测量时间、数值、症状

### 禁忌

- **不要自行加倍服药**
- **不要舌下含服硝苯地平**（可能导致血压骤降）
- **不要过度紧张**（会进一步升高血压）

## 低血压

### 定义

收缩压<90 mmHg和/或舒张压<60 mmHg。

### 症状性低血压

血压低伴以下症状：
- 头晕、黑矇、乏力
- 冷汗、心悸
- 恶心、呕吐
- 晕厥或接近晕厥

### 常见原因

1. **药物相关**
   - 降压药过量
   - 多种降压药联合
   - 其他药物（利尿剂、血管扩张剂）

2. **容量不足**
   - 脱水、大量出汗
   - 腹泻、呕吐
   - 出血

3. **心脏原因**
   - 心律失常
   - 心肌梗死
   - 心力衰竭

4. **体位性低血压**
   - 从卧位/坐位站起时血压下降>20/10 mmHg
   - 常见于老年人、糖尿病患者

### 处理

**症状性低血压应立即处理**：

1. 平卧，抬高下肢
2. 补充水分（如能饮水）
3. 停用可能导致低血压的药物（咨询医生）
4. 若持续不缓解或反复发作，立即就医

### 体位性低血压的预防

- 起床时"三部曲"：躺30秒→坐30秒→站30秒→走
- 避免长时间卧床后突然站起
- 保证充足水分摄入
- 穿弹力袜

## 心率异常

### 需要就医的情况

1. 静息心率>120次/分钟，伴头晕、胸闷
2. 静息心率<50次/分钟，伴乏力、黑矇
3. 心悸持续数分钟以上，无明确诱因
4. 心跳不规则、漏跳感频繁

### 偶发心悸

- 数秒内的心悸可能与早搏有关
- 若偶发、无症状，可观察
- 若频繁发作或伴其他症状，需就医

## 家庭应急准备

建议高血压患者家庭备有：

1. 血压计（定期校准）
2. 急救电话卡片（写明病史、用药）
3. 症状识别指南

### 就医指征总结

| 情况 | 紧急程度 | 建议 |
|------|---------|------|
| 血压≥180/120 + 靶器官损害症状 | 紧急 | 立即就医/拨打120 |
| 血压≥180/120 + 无症状 | 较急 | 尽快就医（数小时内） |
| 血压显著波动 | 一般 | 预约门诊 |
| 低血压 + 症状 | 紧急 | 平卧、就医 |
| 心率异常 + 症状 | 较急 | 就医评估 |
