# 高血压的定义与诊断标准

## 定义

高血压是以体循环动脉血压持续升高为主要临床表现的心血管综合征。血压由收缩压（SBP）和舒张压（DBP）两个数值表示，单位为毫米汞柱（mmHg）。

## 诊断标准

### 诊室血压诊断标准

在未使用降压药物的情况下，**非同日3次测量诊室血压，收缩压≥140 mmHg和/或舒张压≥90 mmHg**，可诊断为高血压。

### 不同测量方法的诊断阈值

| 测量方法 | 收缩压（mmHg） | 舒张压（mmHg） |
|---------|---------------|---------------|
| 诊室血压 | ≥140 | ≥90 |
| 家庭血压 | ≥135 | ≥85 |
| 24小时动态血压（平均值） | ≥130 | ≥80 |
| 白天动态血压 | ≥135 | ≥85 |
| 夜间动态血压 | ≥120 | ≥70 |

## 血压测量规范

### 诊室血压测量

1. 被测量者至少安静休息5分钟
2. 坐位，背靠椅背，双脚平放地面
3. 裸露上臂，绑袖带，袖带下缘距肘窝2-3cm
4. 首诊测量双侧上臂血压，以较高一侧为准
5. 每次测量2-3遍，间隔1-2分钟，取平均值

### 家庭血压测量

1. 每天早晚各测量2-3次，间隔1分钟
2. 早晨：起床后1小时内，服降压药前，早餐前
3. 晚上：晚饭后、睡觉前
4. 初诊或血压未达标者，连续测量7天

## 白大衣高血压与隐蔽性高血压

- **白大衣高血压**：诊室血压升高，但诊室外血压正常
- **隐蔽性高血压**：诊室血压正常，但诊室外血压升高

这两种情况均需要通过家庭血压监测或动态血压监测来确诊。

## 重要提示

血压具有波动性，受情绪、活动、饮食、季节等因素影响。单次测量偏高不能确诊高血压，应以多次测量结果综合判断。若血压持续升高，应及时就医，由专业医生进行评估和诊断。
