## 3. 端到端集成测试（新增）

- 文件：[backend/tests/test_e2e_full_flow.py](../backend/tests/test_e2e_full_flow.py)
- 链路：**注册 → 登录 → /users/me 验 token → OCR（mock VLM）→ 入库 → 看板 list/stats → 创建会话 → SSE 问诊（mock LLM）→ 消息持久化 + 引用落库 → 清理**
- 用例：
  - `test_full_user_journey`（9 阶段 19 项断言）
  - `test_unauthenticated_cannot_cross_modules`（5 个业务接口 401 校验）
- 结果：**2 passed in 0.65s**

外部依赖一律走 fixture mock：
- `ocr_service.recognize` → 返回固定 `OCRFields(135/88/78)` 与 3 个候选
- `chat_service.answer_stream` → 返回固定 deltas + 1 条引用（标题"高血压定义与分级"）

意义：把 P1–P5 各阶段单测**粘成一条路径**，一旦模块间的 schema/鉴权/隔离契约漂移，立刻在此用例红掉。
