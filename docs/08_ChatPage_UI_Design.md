## 组件解剖

### 消息气泡

```
┌─────────────────────────────────┐
│ [Avatar] ┌──────────────────┐   │ ← 助手（绿色机器人头像）
│          │ Markdown 内容   │   │   气泡圆角：16px 16px 16px 4px（右下尖）
│          └──────────────────┘   │
│                └─ 时间戳（11px，40px缩进） │
└─────────────────────────────────┘

┌─────────────────────────────────┐
│        ┌──────────────────┐ [Avatar]─┐ ← 用户（蓝色头像）
│        │ Markdown 内容    │          │
│        └──────────────────┘          │
└─────────────────────────────────┘
```

- **用户气泡**：`border-radius: 16px 16px 4px 16px`（左下尖）
- **助手气泡**：`border-radius: 16px 16px 16px 4px`（右下尖）
- **头像**：32x32，圆形，助手绿/用户蓝
- **时间戳**：11px，置于气泡下方，助手对齐头像，右侧padding 40px

### 输入区域

```
┌──────────────────────────────────────────────────────────┐
│  ┌────────────────────────────────┐  ┌────────┐  ┌────┐  │
│  │ TextArea (borderRadius:24)    │  │ Cancel │  │Send│  │
│  └────────────────────────────────┘  └────────┘  └────┘  │
└──────────────────────────────────────────────────────────┘
              ↑ 分隔线：2px solid #f0f5ff
```

- **输入框**：药丸形状（borderRadius: 24），白色背景
- **发送按钮**：药丸形状，height: 36
- **取消按钮**：圆角20，红色文字

### 快捷问题标签

```
┌────────────┐ ┌────────────┐ ┌────────────┐
│  解读我最近 14 天的血压情况  │  高血压在家如何正确测量  │ ...
└────────────┘ └────────────┘ └────────────┘
```

- 药丸形状（borderRadius: 20）
- 悬停：上浮2px + 阴影 `rgba(47,125,255,0.15)`

## 设计决策

1. **头像系统**：用户=蓝色+UserOutlined，助手=绿色+RobotOutlined，一眼区分
2. **气泡不对称圆角**：用户左下尖/助手右下尖，模拟真实对话感
3. **时间戳置于气泡下方**：不干扰内容，仅11px muted色
4. **脉冲dot替代▍**：CSS动画跨平台一致性优于font字符
5. **引用卡片绿色Tag**：与助手头像颜色呼应，视觉统一
